What is Striverr?

  1. Striverr: Empowering growth, celebrating achievements, connecting journeys, inspiring brighter futures.
  2. Users can login and access many features on the platform focused on self improvement.
  3. Features include an achievements posting page, a challenges posting page, a communicative AI chatbot, a step tracker, a hobbies log, a bucket list recorder, and a log of motivational quotes.
  4. The application is built using Flask, a Python web framework, and uses a PostgreSQL database to store user information and data.
